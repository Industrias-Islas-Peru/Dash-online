# Dashboard Gerencial SCADA

Archivos listos para publicar en GitHub Pages.

## Subir a GitHub

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos a la raiz del repositorio:
   - `index.html`
   - `industrias-islas-logo.png`
   - `chart.umd.min.js`
   - `html2canvas.min.js`
3. En GitHub, entra a `Settings > Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda. GitHub mostrara el enlace publico en unos minutos.

## Nota

La version publicada lee el CSV publico de Google Sheets. El boton de captura descarga el PNG desde el navegador.
